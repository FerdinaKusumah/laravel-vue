## Example branch name convention
    * do:
        * feature/branchName -> camelCase
        * feature/branch-name -> dash | hypen
        * feature/branch_name -> snake case
            -> feature/fooBar
            -> bugFix/fooBar
            -> hotFix/fooBar
    * dont:
        * feature_branch+name
        * feature branch name
        * feature_branch_name_
        * _feature_branch_name
